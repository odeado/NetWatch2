+ Importar directorio completo (Excel/archivo exportado de otro sistema)

Sube un directorio de empleados ya escrito a mano (correo, cargo, VPN, etc). Si el nombre ya esta en el sistema, solo completa los campos que esten vacios -- nunca pisa lo que ya tengas cargado. Si el nombre no existe, crea el empleado nuevo. Sirve para este archivo y para cualquier otro que traigas a futuro con las mismas columnas (Nombre Completo, Email, Departamento, Ciudad, Telefono, Lugar de Trabajo, VPN Activa, Tipo de VPN, Cargo, Sistemas Autorizados).

+ Importar desde otro sistema de gestion (usuarios y equipos, archivo .xlsx)

Para archivos .xlsx reales de otro sistema (hojas "Usuarios", "Equipos" y "Departamentos"), como el que exporta el sistema de gestion de usuarios/equipos. Matchea usuarios por nombre y equipos por IP (si tiene una real) o por hostname si es "Dinamica" (notebook remoto/VPN). Solo completa los campos que esten vacios en tu inventario actual -- nunca pisa ni borra lo que ya tengas cargado en NetWatch. Si un equipo tiene varios usuarios asignados, el primero queda como responsable y el resto se anota en las notas del equipo para no perder el dato.
